Option 1: Qt Creator / MinGW package

Downloads

Qt 5.4.2 for Windows 32-bit (MinGW 4.9.1, OpenGL, 842 MB)

Description

Qt is a popular library built on top of the C++ programming language which provides high level, platform-independent functionality. This package includes the Qt Creator IDE (similar to Eclipse CDT) and the MinGW toolchain in an easily installable form. This can act as a substitute for the Eclipse CDT and MinGW packages.
