Gerrit Tutorial
How to upload code to Gerrit

Instructions

Visit Gerrit and sign in/sign up

gerrit.frc3512.com
Create an SSH Key pair: 
ssh-keygen -t rsa -C "your_email@youremail.com"
Copy your public key and add it to Gerrit: 
cat /home//.ssh/id_rsa.pub

(replace with your msys2 username, to find it, use the command whoami) Copy the entire block, then navigate to the SSH Public Key settings of your user settings on the Gerrit website. Click "Add Key" and paste the entire key, then click "Add"

Installing git-review: 
pacman -S python

wget https://bootstrap.pypa.io/get-pip.py

python get-pip.py

pip install --user git-review
Setting up git-review: cd to root directory of the local repositrory(i.e. /home//Robot-2017), then run git review -s
