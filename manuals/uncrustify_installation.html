Uncrustify Installation

Keys:

$ means in terminal (don't type $ in though)
  1. Go here for getting uncrustify.
  2. Download the *.tar.gz file, check with your lead for the newest version we accept.
  3. $ tar -xf uncrustify-0.61.tar.gz
  4. $ cd uncrustify-*/
  5. $ ./configure
  6. $ make
  7. $ sudo make install

    8. To use uncrustify:
    Go to the directory you want(Uncrustify will format everything in the directory)
    $ ./format.sh
