Setting up WiFi based cameras

  • Find the IP
    • Try a previous IP
    • Use the tool to find it
    • Use a ping tool
      • If all else fails troubleshoot, most likely the cameras not connected to the WiFi (use the amcrest app to connect it) refer to the data portion for more info about the cameras.
  • Open VLC and go to Open Stream in the task bar
    • Use rtsp://username:password@IP-Address(thanks Will) to connect start the stream.
  • Ensure that the cameras are oriented correctly and video quality settings are okay
    • In the event there are video quality or bandwidth issues feel free to edit the camera settings if you know how to and are confident in what your doing
