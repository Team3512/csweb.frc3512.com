¿Dónde está mi calculadora?

I've lost my calculator and I've got an important test in Calculus 2 coming up. I need you to make me a temporary calculator that I can use to study for test. It will need to do derivatives, integrals, summations, graphing and... oh yea. Maybe we can just start with the basics; addition, subtraction, multiplication and division.

You should create a simple command line calculator in C++. Use a while loop to take user inputs until the user types q(uit). Take an input of 2 floating point numbers and a character. The character will determine the operation to be used on the numbers. I only want the answer to have a decimal precision of two places. Don't fail me. I'm heavily relying on this calculator and I wont hesitate to blame you for my inability to do simple operations during the test.
