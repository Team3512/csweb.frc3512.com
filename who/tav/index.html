Modules

Computer Science 101 (Basics of C++)

Introduces the fundamentals of computer programming and software design. Topics include variables, data types, assignment, expressions, basic I/O, control flow, functions and parameters, scope, and data structures.

Computer Science 102 (Threading and Networking)

Covers more advanced topics such as threading, synchronization, and asynchronous data passing (i.e. networking). Proper use of C++11 threads, mutexes, and atomics will be taught. SFML is used to instruct students in the proper use of UDP and TCP sockets, as well as when to use each type.

Computer Science 103 (Shell Scripting)

An introductory course in writing shell scripts using Bash. Shell scripts will be used to automate sequences of commands to optimize parts of one's workflow.

Computer Science 104 (HTML)

Introduces website design using HTML, CSS and Javascript. Emphasis is placed on writing HTML5 conformant web pages.

Computer Science 105 (Vision Processing)

Teaches basic vision processing using the OpenCV computer vision library.

Computer Science 106 (Perceptrons and Artificial Neural Networks)

Introduces the field of machine learning with the perceptron. A basic example in vision processing is presented.

Computer Science 107 (Android Development)

Introduces writing applications using the Android SDK. Instruction in the Java programming language is provided as needed.

Git 101 (Basics of Git)

Introduces version control systems and their uses. This course is taught using Git. Installation of Git, setting up user identification for commits, and basic workflow are covered, with emphasis on the use of branches to manage work between repositories.

Git 102 (Advanced Git)

Expands on the set of commands and techniques learned from the previous course. Topics include tags, rebasing, and managing repository history.

Control Theory 101 (PID Controllers)

Introduces the topic of control systems and how they can improve the performance of a robot. PID controllers are presented as a simple yet effective tool for most control applications in FRC. Their dynamics, types, and procedures for tuning them effectively are discussed.

Control Theory 102 (Motion Profiling)

This is a continuation of Control Theory 101. PID controllers are used in conjunction with motion profiles of several types to produce more consistent system responses. The implementation of motion profiles as well as tuning PID controllers which don't undergo step responses are covered.

Signal Processing 101 (Filters and State Estimation)

Discusses filters, why they are necessary, and their application to control theory. Several basic filters are covered such as the rolling average filter, complimentary filter, low-pass filter, and high-pass filter. Each one is studied for its respective use in predicting the state of a system.

Signal Processing 102 (Kalman Filter and Linear Algebra)

This course is devote entirely to the Kalman filter. Its inner workings as well as its utility in state estimation will be discussed. A single-input Kalman filter will be used for simplicity. Topics include noise distribution and some linear algebra.

FRC 101 (Scouting Application)

Focuses on writing a software application for scouting during FRC competitions. This has typically been written in conjunction with a spreadsheet which receives the scouting data. HTML web pages and Android applications have been used in the past.
