Developer Bootstrap

This page provides setup instructions for the software team's tools so members can quickly start learning C++. The installation instructions target Microsoft Windows. For Linux machines, install Eclipse CDT, JDK 8, GCC, Git, and an LLVM package containing clang-format 3.8.1 or newer via that machine's package manager.

Eclipse CDT

Downloads

Eclipse IDE for C/C++ Developers
JDK 8

Description

Eclipse CDT is an integrated development environment which makes it easier to write, build, and test programs written in C and C++.

Installation Instructions

On the Eclipse CDT download page, select the appropriate platform on the page's right. If your machine is 64 bit, choose a 64 bit version. Once the download is complete, extract it to a directory of your choice.

On the JDK download page, select the JDK installer for the same operating system and architecture of Eclipse CDT, download it, and run it.

MSYS2 Compiler

Downloads

https://msys2.github.io/

Description

This is the compiler Eclipse CDT will use to build the programs you write.

Installation Instructions

Follow the instructions on the web page. Use the default installation directory.

When installation is complete and everything has been upgraded with pacman, add "C:\msys64\usr\bin" or "C:\msys32\usr\bin" to your system-wide PATH environment variable depending on whether you used the x86_64 installer or i686 installer respectively.

You may want to create a desktop shortcut for “MSYS2 MSYS” to facilitate opening MSYS2 terminals. Open an MSYS2 terminal and run the following command to install make, gcc, and other required tools.

pacman -S base-devel gcc

Learning C++

Once the development environment is set up, one may begin learning C++. We use the textbook Programming: Principles and Practice Using C++ (2nd edition) by Bjarne Stroustrup (ISBN: 978-0-321-99278-9). Either a copy will be provided by the software team or a copy can be bought directly from the publisher. The current textbook errata are here.

Chapter 2 of the book requires std_lib_facilities.h. Useful resources for the C++ standard library include http://en.cppreference.com/w/cpp.

Formatter

We use the Python 3 scripts provided in WPILib's style guide repository and clang-format for source code formatting. We also use Python 3 for optimizing feedback control system behavior.

LLVM

Installation Instructions

On the LLVM download page, select the installer for latest version of LLVM corresponding to your operating system and architecture under "Pre-Built Binaries", then run it. The default installation options are fine.

MSYS2 keeps a separate PATH environment variable from the operating system. Run the following command to add the LLVM installation to it.

echo 'export PATH=$PATH:/c/Program\ Files/LLVM/bin' >> .bashrc

The following commands download the WPILib style guide, configure its environment variable, and install Python.

git clone git://github.com/wpilibsuite/styleguide $HOME/styleguide
echo 'export WPI_FORMAT=$HOME/styleguide' >> .bashrc
pacman -S python

After those commands complete, close and reopen the terminal for the environment variable changes to take effect.
