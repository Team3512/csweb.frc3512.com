Topic 8 - Pointers

Topic 8 Slides

Question 1

Print the numbers 33 to 126 and their corresponding ASCII character values.

Question 2

Prompt the user for a string. Output the length of the string.

Question 3

Prompt the user for a string. Output the string in reverse.

Question 4

Prompt the user for a line of text. Output that line in all capitals. Leave non-alphabetic characters untouched.

Question 5

Prompt the user for two strings, s1 and s2. Determine if s1 and s2 are equal. If they're not equal, output them in alphabetical order.

Question 6

Prompt the user for his/her name. Verify that the name only contains alphabetic characters.

Question 7

Prompt the user for a social security number. Determine if the SSN is in the correct format.

Question 8

Determine if a word entered by the user is spelled correctly. A word is considered correct if it's found in dict.txt.
