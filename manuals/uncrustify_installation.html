Uncrustify Installation

hr>
  1. Go here for getting uncrustify.
  2. Download the *.tar.gz file, check with your lead for the newest version we accept.
  3. $ tar -xf uncrustify-0.62.tar.gz
  4. $ cd uncrustify-*/
  5. $ ./configure DESTDIR=/home/[user]/uncrustify-bin
  6. $ make
  7. $ make install

    To use uncrustify:
    • Go to the directory you want(Uncrustify will format everything in the directory)
    • $ ./format.sh
    • If you are not in the directory, type $ pwd
    • Copy the output then put that in before running $ ./format.sh
