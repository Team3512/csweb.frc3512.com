Option 2.2: MSYS2 Compiler

Downloads

https://msys2.github.io/

Description

This is the compiler Eclipse CDT will use to build the programs you write.

Installation Instructions

Follow the instruction on the web page. Use the default installation directory. When installation is complete and everything has been upgraded with pacman, Add "C:\msys64\usr\bin" or "C:\msys32\usr\bin" to your PATH environment variable depending on whether you used the x86_64 installer or i686 installer respectively.

To actually install the compiler into the environment, open an MSYS2 terminal and run "pacman -S base-devel". This will install make, gcc, and other required tools.
