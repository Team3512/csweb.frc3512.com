Setting up new computers to FRC Team 3512's standards

Keys:

$ means in terminal (don't type that in though)

  1. Make sure they have ubuntu is installed
  2. $ sudo apt-get upgrade
  3. $ sudo apt-get update
  4. If terminal shows any old or corrupted files, do $ sudo apt-get dist-upgrade
  5. To install eclipse, follow this guide here.
    6. Launch the installer rather than doing all of the tar stuff.
    Select the "Eclipse IDE for C/C++ Developers" and install that.
  6. Install the plugin helper here.
  7. Goto the tool chain link here. and follow the instructions.
  8. Open eclipse, goto "Help"
    Press "Install new sofware"
    Press "Add" then name it
    Then put this as the location: "http://first.wpi.edu/FRC/roborio/release/eclipse/"
  9. To install git follow these instructions here.
  10. Install uncrustify, go to the Uncrustify Installation manual for help.
