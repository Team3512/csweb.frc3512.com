Setting up new computers to FRC Team 3512's standards

Keys:

$ means in terminal (don't type that in though)

  1. This manual is only applicable to Debian, Ubuntu and Xubuntu!
  2. $ sudo apt-get update
  3. $ sudo apt-get dist-upgrade
  4. To install eclipse, download the version, corresponding to your computer here.
    $ tar -xf "tarfile"
  5. Install the plugin helper here.
  6. Goto the toolchain link here. and follow the instructions.
  7. Open eclipse, goto "Help"
    Press "Install new sofware"
    Press "Add" then name it
    Then put this as the location: "http://first.wpi.edu/FRC/roborio/release/eclipse/"
  8. To install git follow these instructions here.
  9. Install uncrustify, go to the Uncrustify Installation manual for help.
