Uncrustify Installation

  1. Go here for getting uncrustify.
  2. Download the *.tar.gz file. Check with your lead for the newest version we accept.
  3. $ cd to your downloads folder.
  4. $ tar -xzf uncrustify-*.tar.gz
  5. $ cd uncrustify-*/
  6. $ ./configure
  7. $ make
  8. $ sudo make install prefix=/home/[user]/uncrustify-bin"
  9. To check that it installed correctly type $ uncrustify -v If there are errors, contact your lead or try this again.

    To use uncrustify:
    • Go to the project directory containg format.sh (Uncrustify will format everything in the directory)
    • $ PATH=$PATH:/home/[user]/uncrustify-bin ./format.sh
    • cd to the folder of the robot and then format using ./format.sh
    • i.e: username@usernamedirectory VirtualBox:~/git/Robot-2016$ ./format.sh
      Your command line should look like this before you run ./format.sh
