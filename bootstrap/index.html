MSYS2 Compiler

Downloads

https://msys2.github.io/

Description

This is the compiler Eclipse CDT will use to build the programs you write.

Installation Instructions

Follow the instruction on the web page. Use the default installation directory. When installation is complete and everything has been upgraded with pacman, Add "C:\msys64\usr\bin" or "C:\msys32\usr\bin" to your PATH environment variable depending on whether you used the x86_64 installer or i686 installer respectively.

To actually install the compiler into the environment, open an MSYS2 terminal and run pacman -S base-devel gcc. This will install make, gcc, and other required tools.

Learning C++

Once the development environment is set up, one may begin learning C++. We use the textbook Programming: Principles and Practice Using C++ (2nd edition) by Bjarne Stroustrup (ISBN: 978-0-321-99278-9). Either a copy will be provided by the software team or a copy can be bought directly from the publisher. The current textbook errata are here.

Useful resources for the C++ standard library include http://en.cppreference.com/w/cpp.
